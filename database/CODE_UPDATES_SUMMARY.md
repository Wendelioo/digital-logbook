Complete profile photo management system using file storage instead of database BLOBs.

- MIME type validation (JPEG, PNG, GIF)
- Automatic cleanup on database insert failure
- Transaction safety (delete file if DB insert fails)

**Constants:**

```go
)
```

## Migration Checklist

### Pre-Migration
- [ ] Backup database: `mysqldump -u root -p logbookdb > backup.sql`
- [ ] Create `uploads/profiles/` directory
- [ ] Set directory permissions: `chmod 755 uploads/profiles/`
- [ ] Stop application (optional, for zero downtime)

### Migration Execution
- [ ] Run migration script: `mysql -u root -p logbookdb < migration_to_improved_schema.sql`
- [ ] Verify new tables/columns created

================================================================================
                         USE CASE DIAGRAM GUIDE
                         Digital Logbook System
================================================================================

A Use Case Diagram shows the interactions between actors (users) and the
system, identifying what functions each actor can perform.


================================================================================
                              ACTORS
================================================================================

1. Admin           - System administrator with full access
2. Teacher         - Faculty member managing classes and attendance
3. Student         - Regular student using PC lab
4. Working Student - Student employee with monitoring duties


================================================================================
                         USE CASES BY ACTOR
================================================================================


ADMIN
--------------------------------------------------------------------------------

  Authentication:
    - Login
    - Logout
    - Change Password

  User Management:
    - Create User
    - Edit User
    - Delete User
    - Search Users
    - View All Users
    - Filter Users by Type
    - Bulk Import Users from CSV
    - Approve Pending Registrations
    - Reject Pending Registrations

  Profile:
    - View Profile
    - Edit Profile
    - Upload Profile Photo
    - Delete Profile Photo

  Department Management:
    - Create Department
    - Edit Department
    - Delete Department
    - View Departments

  Class Management:
    - View All Classes
    - Create Class
    - Edit Class
    - Delete Class
    - Activate/Deactivate Class
    - Archive/Unarchive Class

  Login Logs:
    - View All Logs
    - View Today's Logs
    - View Past Logs (by Date Range)
    - Archive Logs by Selection
    - Archive Logs by Date
    - View Archived Logs
    - Unarchive Logs

  Feedback:
    - View All Feedback
    - View Archived Feedback
    - Archive Feedback
    - Unarchive Feedback

  Reporting:
    - Export Logs to CSV
    - Export Logs to PDF
    - Export Feedback to CSV
    - Export Feedback to PDF
    - Export Archived Logs to CSV/PDF
    - Export Archived Feedback to CSV/PDF

  Dashboard:
    - View Dashboard Statistics
